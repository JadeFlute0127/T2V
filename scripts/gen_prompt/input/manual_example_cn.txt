向澄清石灰水吹气变浑浊实验说明书（化学-无机化学）
一、实验基本信息
1. 实验名称：向澄清石灰水吹气变浑浊实验
2. 所属学科：化学-无机化学
3. 实验目的：通过直观实验现象验证二氧化碳与氢氧化钙的化学反应，理解碱与酸性氧化物的反应规律；掌握澄清石灰水的取用、实验现象观察及实验原理总结的基本方法。
二、实验器材
试管（1支，规格15×150mm）、试管架、胶头滴管、玻璃导管（带橡胶管，长度30cm左右）、澄清石灰水（饱和氢氧化钙溶液，无色透明）、烧杯（用于放置导管，可选）、护目镜（必备，保障实验安全）。
三、实验操作流程说明
准备阶段：将试管洗净后放置在试管架上，佩戴好护目镜；用胶头滴管吸取澄清石灰水，向试管内注入1/3体积的澄清石灰水，确保溶液无沉淀、无浑浊，状态均匀。
器材连接：将玻璃导管的一端插入试管内的澄清石灰水中（导管末端置于液面下，避免触碰试管壁），另一端手持备用；若使用烧杯，可将导管非插入端放置在烧杯上方，便于操作稳定。
核心操作：用嘴对准玻璃导管的外露端，缓慢、匀速地向澄清石灰水中吹气，持续吹气过程中保持导管末端始终在液面下，避免空气未充分接触溶液。
现象观察：吹气过程中密切关注试管内溶液的状态变化，直至观察到明显现象后停止吹气。
实验收尾：先将导管从试管内取出，再停止吹气（防止溶液倒吸）；将试管仍置于试管架上供观察，实验结束后将溶液倒入指定废液桶，清洗试管及其他器材并归位。
四、实验现象观察
1. 初始状态：试管内的澄清石灰水为无色透明液体，无任何浑浊或沉淀。
2. 吹气过程中：随着持续吹气，无色透明的澄清石灰水逐渐出现浑浊现象，浑浊程度随吹气时间增加而加深，最终整个溶液变为乳白色浑浊状态。
3. 过量吹气（可选）：若持续大量吹气，乳白色浑浊溶液会逐渐变澄清（此为补充现象，核心现象为吹气后变浑浊）。
五、实验原理总结
本实验的核心原理是无机化学中碱与酸性氧化物的反应，具体为二氧化碳（酸性氧化物）与氢氧化钙（碱）反应生成难溶性的碳酸钙沉淀，使溶液变浑浊。
1. 核心反应方程式：Ca(OH)₂ + CO₂ = CaCO₃↓ + H₂O
2. 原理分析：人体呼出的气体中含有约4%-5%的二氧化碳，当二氧化碳通过导管进入澄清石灰水（主要成分为Ca(OH)₂）时，二者发生化学反应，生成的碳酸钙（CaCO₃）是一种难溶于水的白色固体，因此溶液会出现乳白色浑浊现象。
3. 补充原理（过量吹气）：若持续通入过量二氧化碳，生成的碳酸钙会与二氧化碳、水进一步反应，生成可溶于水的碳酸氢钙（Ca(HCO)₂），因此浑浊溶液会重新变澄清，反应方程式为：CaCO₃ + CO₂ + H₂O = Ca(HCO₃)₂（此为拓展原理，核心实验原理聚焦于前者）。
六、实验注意事项
1. 实验前需检查澄清石灰水状态，确保为无色透明，若已浑浊则不可使用（可能已变质）。
2. 吹气时需缓慢匀速，避免用力过猛导致溶液溅出或导管脱落；同时确保导管末端始终在液面下，保证气体充分接触溶液。
3. 实验结束后需先取出导管再停止吹气，防止试管内压强减小导致溶液倒吸进入呼吸道，引发安全隐患。
4. 实验废液需倒入指定容器，不可随意排放；器材清洗后及时归位，保持实验台整洁。
